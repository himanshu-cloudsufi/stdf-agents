Tony does not agree with LCOE as the primary cost metric for energy sector disruption analysis. He prefers the "marginal cost kill condition": the threshold where it is cheaper to build new SWB from scratch than to pay the fuel bill on an existing, fully-depreciated incumbent unit.

**Why:** LCOE includes amortized capital for the incumbent, which has already been paid. The decisive economic question is whether a rational operator would keep running an existing incumbent or switch to new SWB. That comparison is SWB amortized capex vs. incumbent marginal fuel+O&M only.

**How to apply:**
- Always compute BOTH thresholds: (1) new-vs-new LCOE parity, (2) new SWB vs. existing incumbent marginal cost
- For criterion 5.3 tipping condition (commercial displacement signal): use LCOE parity — this is when new orders collapse
- For existential endpoint (installed-base retirement signal): use marginal cost kill
- Report both explicitly as Threshold 1 and Threshold 2 in a dual-threshold table
- Never use LCOE alone without reporting the marginal cost kill separately
- Label NG price sensitivity: LCOE parity is NG-insensitive (capital-dominated); marginal kill is highly NG-sensitive
