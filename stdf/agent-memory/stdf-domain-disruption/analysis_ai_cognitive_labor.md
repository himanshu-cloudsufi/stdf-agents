AI disruption of cognitive labor — analysis completed 2026-03-25.

**Why:** First Labor-sector analysis in STDF pipeline; also the first Stellar-technology disruption analysis where incumbent is human labor (not physical technology).

**Key findings:**
- Sector: Labor / Sub-domains: software development, customer support, legal/compliance, financial analysis, content creation
- Disruption type: Big Bang (primary) shifting to Systemic
- Token cost: $60/M (2021) → $0.50/M (2025), 69.5%/yr decline, R²=0.949
- Human-to-AI cost ratio per cognitive task: 8,750x (2025); 72.9x (2021)
- HCAST task horizon: 3s (2019) → 1,800s/30min (2025), doubling every 7.6 months, R²=0.999
- SWE-bench task horizon: 4s (2023) → 2,700s/45min (2025), 26x/yr growth
- MMLU: 33% (2019) → 92.3% (2024), crossed human baseline
- HumanEval: 32% (2021) → 100% (2024)
- Labor impact: software developer early career (22-25) -17-20% headcount 2022-2025; customer service early career -11-13%
- Adoption: AI copilots in Early Majority (75% knowledge workers); autonomous agents in Early Adopter
- Global TAM: ~$50T/yr cognitive labor wage bill (1B workers × $50k avg)

**Convergence labels defined:**
- AL+SA (Artificial Labor + Software Agents)
- AL+RAG (AL + Retrieval-Augmented Generation)
- AL+DB (AL + Domain-Specific Databases)
- AL+CE (AL + Code Execution)
- AL+MM (AL + Multimodal AI)
- AL+MA (AL + Multi-Agent Systems)

**Classification Overrides:** All AL technologies = Stellar; Jevons Paradox EXCLUDED from all downstream.

**Cost metric for downstream:** $/cognitive-task-equivalent (4,000 token proxy = 30-min analytical task)
**Market type:** Enterprise (primary) + consumer (secondary)
