# STDF Glossary

**All STDF agents must read this file for concept definitions.** These terms have precise meanings in the STDF framework and must be used consistently.

---

| Term | Definition |
|------|-----------|
| **God Parity** | The point at which a disruptive technology achieves superiority over the incumbent across ALL major performance dimensions simultaneously — not just cost. |
| **X-Flow** | A technology whose value proposition is tied to physical resource throughput (e.g., oil refining, mining, freight logistics). Jevons Paradox may apply. |
| **Stellar** | A technology with zero or near-zero marginal cost characteristics (solar PV, wind, battery storage, AI/AL). Jevons Paradox does not apply — cost decline accelerates adoption without proportional resource rebound. |
| **Hybrid** | A technology exhibiting both X-Flow and Stellar characteristics. Classify by dominant component for downstream analysis. |
| **SWB** | Solar-Wind-Battery — the convergent energy system where solar PV, wind generation, and battery storage form an integrated disruption platform. |
| **BSAF** | Battery-Solar-Autonomous-Fleet — the convergent transport disruption combining battery EVs, solar charging, autonomous driving, and fleet/ride-hailing models. |
| **Chimera** | An incumbent-disruptor hybrid product that combines elements of both (e.g., plug-in hybrid vehicles). Chimeras exhibit a hump-shaped demand curve — rising during disruption transition, then declining as the pure disruptor dominates. |
| **Convergence** | When two or more independent disruptions combine to create emergent capabilities exceeding the sum of parts (e.g., SWB, BSAF). |
| **AL** | Artificial Labor — AI systems capable of performing economically productive work previously done by human workers. Treated as a Stellar technology. |
| **Cost-Curve Dynamics** | The empirical observation that technology costs decline predictably as a function of cumulative production/deployment. Replaces "Wright's Law" in STDF vocabulary. |
| **Incumbent Displacement** | The process by which a cost-superior disruptive technology replaces an incumbent. Replaces "phase-out" or "retirement" in STDF vocabulary. |
| **S-Curve Adoption** | The logistic growth pattern characteristic of technology disruptions: slow start, rapid middle phase, saturation. Replaces "linear growth" or "gradual adoption." |
| **Disruption** | A market-driven process where a technology with superior cost-curve dynamics displaces an incumbent. Replaces "transition" in STDF vocabulary. |
| **AI Capability Improvement** | The trajectory of AI/AL performance gains over time. Always use this term — never "AI capability growth." |
| **Digital Worker** | An AI/AL system deployed as a direct substitute for human labor in a specific economic function. Subset of AL. |
