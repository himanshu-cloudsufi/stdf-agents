Design Principle

STDF Agents makes the framework the architecture. Cost curves must be fitted before tipping points can be assessed. Tipping must be synthesized before S-curves can be modeled. Each agent reads upstream output files — there is no way to skip steps. The pipeline order is the Seba framework.
