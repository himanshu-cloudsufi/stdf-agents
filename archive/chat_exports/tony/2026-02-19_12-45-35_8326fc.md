**Key Cost-Rupture Drivers**

1. **Sensor Fusion Architecture**: Multi-sensor integration reduces per-sensor cost via shared compute resources and calibration infrastructure [C39]. Centralized domain controllers replace distributed ECUs, reducing total system cost by 20-30% [C40].

2. **Semiconductor Node Transitions**: 7nm (2024-2026) → 5nm (2027-2029) → 3nm (2030-2033) [C41]. Each node transition delivers 25-35% cost reduction and 40-50% power reduction, enabling higher compute density without thermal/power constraints [C42].

3. **Standardization & Commoditization**: Industry-wide adoption of common interfaces (MIPI CSI-2 camera, CAN FD, Automotive Ethernet) enables commodity pricing and multi-sourcing [C43]. Standardized 77 GHz radar front-end modules reduce supplier lock-in [C44].

4. **China Manufacturing Scale**: Domestic Chinese radar and camera production creates 30-40% cost advantage vs Western suppliers [C45]. Vertical integration (Huawei [[COMPANY:CO8]], Baidu [[COMPANY:CO9]], DJI [[COMPANY:CO10]]) drives component costs down [C46]. Export-oriented production scales global volumes [C47].

5. **Software-Defined Architecture**: Shift from hardware-defined to software-defined features reduces physical component count by 15-25% [C48]. Enables continuous feature improvement without hardware changes, increasing software licensing revenue [C49].
