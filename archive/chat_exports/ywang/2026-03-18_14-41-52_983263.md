The scarce resource in a world of near-zero AI inference cost is not compute or models; it is **proprietary workflow context, domain-specific data, and regulatory embeddedness**. Software firms that own these assets become more valuable as AI inference approaches zero. Software firms whose entire value proposition is a human workflow that AI agents can now replicate for free face an existential rupture beginning in 2027.

## ServiceNow [[COMPANY:CO1]]: Business Model Overview

**ServiceNow** is an enterprise cloud platform company built on a single, unified architecture called the **Now Platform**. Its core function is workflow automation and orchestration — digitizing, automating, and connecting business processes across an organization's departments and systems.

### Core Business Model

ServiceNow operates as a **pure-play SaaS platform** with subscription-based recurring revenue. Unlike point-solution SaaS (which automates a single function), ServiceNow is a **horizontal workflow platform** that spans multiple enterprise functions from a single data model and UI. The platform's value proposition is cross-departmental process orchestration — connecting IT, HR, Finance, Legal, and Operations workflows into a unified system of record.
